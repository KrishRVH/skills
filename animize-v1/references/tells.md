# Tells

The full catalog. Sweep every section against the full text, in order; no section may go unchecked against any part of the text. Each entry pairs the pattern with its fix and, wherever the fix has a shape, a before/after pair: match the text against the befores, produce the shape of the afters. Apply the fix, never just delete the evidence (see Two laws in SKILL.md). Section tags name the verb each tell fails: [performs], [evades], [flattens].

Take the word lists literally: a banned word's synonym is innocent unless listed. Vocabulary tells also drift as models change (see the era table at the end of §1), so the structural sections, §3 onward, outlive any word list; weight them more.

## 1. Words

### Banned outright [performs]
Replace on sight in nonfiction. If the sentence still sounds inflated after the swap, restructure it.

| Banned | Use instead |
|---|---|
| delve | dig into, look at |
| tapestry, symphony, labyrinth, beacon (abstract) | mix, blend, maze, model (or restructure) |
| landscape, realm, ecosystem (abstract) | field, area, space |
| pivotal, paramount | important, or state why it matters |
| underscore, bolster (verbs) | show, support |
| testament (figurative) | proof, sign |
| intricate, intricacies | complicated, details |
| meticulous(ly) | careful(ly), thorough(ly) |
| nuanced, multifaceted | subtle, complex |
| embark, spearhead | start, lead |
| garner | get, earn |
| interplay | interaction |
| leverage, utilize, harness (verbs) | use |
| facilitate, empower, streamline | help, let, simplify |
| foster(ing) | encourage, grow |
| showcase | show |
| boasts (meaning has) | has |
| seamless(ly), effortlessly | smooth(ly), or cut |
| holistic, comprehensive (reflexive) | complete, full |
| cutting-edge, state-of-the-art, groundbreaking (figurative) | new, latest, or the specific advance |
| game-changer, paradigm shift, transformative | say what changed, concretely |
| elevate, supercharge (figurative) | improve, or the specific gain |
| ever-evolving, evolving landscape | changing, or cut |

### Watch list [performs]
Overused rather than banned: at most one instance each, and only where a person would naturally say it. When in doubt, swap.

| Watched | Use instead |
|---|---|
| crucial, vital | important, needed |
| key (adjective) | main |
| vibrant | lively, busy, or the concrete detail |
| robust | solid, reliable |
| enhance | improve |
| valuable | useful |
| enduring | lasting |
| dynamic | active, changing |
| innovative | new, or name the innovation |
| profound | deep, or cut |
| renowned, iconic | well-known, famous |
| notably | delete, or name why it is notable |
| resonate | connect, land |
| illuminate | show, explain |
| align with | match, fit |
| highlight, emphasize (verbs) | point out, stress, or restructure |
| stakeholders | name the actual people |
| synergy | name the actual combined benefit |

### Business jargon [evades]
| Jargon | Use instead |
|---|---|
| navigate (challenges) | handle, deal with |
| unpack | explain, examine |
| lean into | accept, embrace |
| deep dive | analysis, close look |
| double down | commit, increase |
| take a step back | reconsider |
| moving forward | next, from now on |
| circle back | return to, revisit |
| on the same page | agreed, aligned |
| touch base | talk, check in |
| move the needle | make a measurable difference (state it) |
| low-hanging fruit | the easy wins (name them) |

### Promotional puffery [performs]
Kill in any factual register; these turn encyclopedias into travel brochures: nestled, in the heart of, rich cultural heritage, rich history, breathtaking, stunning (natural beauty), must-visit, must-see, world-class, hidden gem, boasts a, commitment to excellence, dedication to craftsmanship, diverse array, captivates residents and visitors alike.
> Before: "Nestled within the breathtaking region of Gonder, Alamata Raya Kobo stands as a vibrant town with a rich cultural heritage and stunning natural beauty."
> After: "Alamata Raya Kobo is a town in the Gonder region of Ethiopia."

### Empty adverbs and intensifiers [evades]
Apply the deletion test: remove the word; if the meaning is unchanged, leave it removed. Usual suspects: really, just, literally, honestly, genuinely, simply, actually, truly, deeply, fundamentally, inherently, incredibly, importantly, crucially, interestingly, undoubtedly, arguably, quietly (as drama).
> Before: "This is quietly reshaping the industry." After: "Three of the five largest insurers adopted it last year."
Keep the ones doing real work: a hedge or intensifier carrying the writer's actual uncertainty or emphasis is human speech ("I honestly don't know", "it tends to fail under load", "perhaps that was the mistake"). Scrubbing every one produces sanitized text, which is its own tell.

### Era drift (for the detect job, and for calibrating the sweep)
| Era | Co-occurring vocabulary |
|---|---|
| 2023 to mid-2024 (GPT-4 class) | delve, tapestry, testament, intricate, meticulous, pivotal, underscore, garner, bolstered, boasts, interplay, vibrant, crucial, Additionally (sentence-initial), enduring |
| Mid-2024 to mid-2025 (GPT-4o class) | align with, fostering, showcasing, highlighting, enhance, enduring, pivotal, vibrant, crucial |
| Mid-2025 on (GPT-5 class) | emphasizing, enhance, highlighting, showcasing, plus the notability-piling phrases of §6 |
| Grok family | empirical, causal, correlate, underscore, and the "X rather than Y" reflex of §3 |
These words cluster: where there is one, expect others. A text dense in one era's set is strong evidence; a text with one stray word is nothing.

## 2. Phrases

### Throat-clearing openers [performs]
Delete and state the point: Here's the thing, Here's what/why/how..., Let me be clear, The truth is, To be honest, I'll be honest, The uncomfortable truth is, It turns out, Let's dive in, Let's unpack this, Let's break this down, Without further ado, What if I told you, Think about it:, Can we talk about, Real talk, and standalone theatrical "Honestly?" or "Look," before an ordinary claim. A person being honest just says the thing.
> Before: "Here's the thing: most bioinformatics pipelines break in production. Not because the code is bad. Because the data is bad. Let that sink in."
> After: "Most bioinformatics pipelines break in production. The code runs fine. The data doesn't match the assumptions baked into it."

### Filler constructions [flattens]
| Filler | Fix |
|---|---|
| It's worth noting that / It's important to note | delete, state the thing |
| In order to | to |
| Due to the fact that | because |
| At this point in time | now |
| In the event that | if |
| For the purpose of | for, to |
| has the ability to | can |
| made a decision | decided |
| At the end of the day, When it comes to, At its core | delete |
| In today's world, In the age of, In a world where | delete |
| The reality is, In terms of, With regard to | delete or restructure |
| a wide range of, a variety of (reflexive) | name two real items |

### Emphasis crutches [performs]
Delete: Full stop. Period. Let that sink in. Read that again. Make no mistake. This matters because. Here's why that matters.

### Faux-insight setups [performs]
"What most people get wrong", "Here's what nobody tells you", "The part everyone misses", "This is the part most people skip". Cut the setup; the claim stands alone.
> Before: "The part everyone misses: distribution is the real moat." After: "Distribution is the moat."

### Persuasive-authority tropes [performs]
"The real question is", "at its core", "what really matters", "the deeper issue", "the heart of the matter". The sentence that follows restates an ordinary point with ceremony.
> Before: "The real question is whether teams can adapt. At its core, what really matters is organizational readiness."
> After: "The question is whether teams can adapt. That mostly depends on whether the organization is ready to change its habits."

### Importance puffery [performs]
stands/serves as a testament, marks a pivotal moment, plays a vital/crucial/significant role, underscores/highlights its importance/significance, reflects broader trends/debates, cements/solidifies its position/reputation, leaves a lasting impact/legacy, indelible mark, key turning point, watershed moment, deeply rooted, setting the stage for, focal point, contributing to the broader. Fix: state the fact; the reader judges significance.
> Before: "The Statistical Institute of Catalonia was officially established in 1989, marking a pivotal moment in the evolution of regional statistics in Spain."
> After: "The Statistical Institute of Catalonia was established in 1989, part of a wider decentralization of administrative functions in Spain."
> Before: "Preserving this endemic species is vital not only for ecological diversity but also for sustaining cultural traditions."
> After: "The species is endemic to Hawaii. The IUCN has not assessed its conservation status."

### Chat and meta artifacts [evades]
Delete every trace of chatbot correspondence and document self-narration: I hope this helps, Certainly!, Of course!, Great question, You're absolutely right, Would you like me to..., Let me know if..., Should I continue?, here is a..., As an AI language model, I cannot offer medical advice but, In this article/section we will..., As we'll see, Let me walk you through, The rest of this essay explains. Also delete unfilled phrasal templates and placeholders: [Your Name], [Insert X here], [link to article], PASTE_URL_HERE, 2025-XX-XX, INSERT_SOURCE_URL, "[Describe the specific section...]".
> Before: "Here is an overview of the French Revolution. I hope this helps! Let me know if you'd like me to expand on any section."
> After: "The French Revolution began in 1789 when financial crisis and food shortages led to widespread unrest."

### Knowledge-cutoff disclaimers and speculative gap-filling [evades]
"as of my last knowledge update", "based on available information", "While specific details are limited/not widely documented...", "in the provided sources". And the gap-fill that follows: "suggesting she maintains a low profile", "keeps personal details private", "prefers to stay out of the spotlight", "likely grew up...". The disclaimer and the guess are both fabrication risks. Fix: say plainly what is not known, or cut the sentence; state a fact only when a source provides it.
> Before: "Information about her early life is not publicly available, suggesting she maintains a low profile. She likely grew up in a middle-class household, which shaped her later interest in education reform."
> After: "Her early life is not documented in the available sources." (Or omit the section.)

### Didactic disclaimers [performs]
"It's important/crucial to note/remember/consider", "worth noting", "may vary". Cut, or state the specific caveat with its specifics attached.
> Before: "It's important to remember that what's free in one country might not be free in another, so always check before you use something."
> After: "Public-domain status varies by country."

## 3. Sentences

### Negative parallelism, every variant [performs]
The single strongest modern structural tell. Variants: not just X but (also) Y; not only X but Y; not X, but Y; It's not X. It's Y.; The question/answer/problem isn't X, it's Y; X isn't the problem, Y is; It feels like X, it's actually Y; no X, no Y, just Z; stops being X and starts being Y; doesn't mean X but actually Y; X rather than Y as a reflex; and clipped tailing negations ("...no guessing.", "...no wasted motion.").
Fix: state Y as a direct claim. If a real misconception needs correcting, name who holds it and why it is wrong.
> Before: "It's not just about the beat riding under the vocals; it's part of the aggression and atmosphere. It's not merely a song, it's a statement."
> After: "The heavy beat adds to the aggressive tone."
> Before: "The question isn't the model. It's the eval." After: "The eval matters more than the model."
> Before: "Not a career, not a body of work, not sustained relevance, just an algorithmic moment." After: "She is known for one viral incident."
> Before (tailing negation): "The options come from the selected item, no guessing."
> After: "The options come from the selected item, so the user never has to guess."

### Negative listing [performs]
> Before: "Not a tool. Not a framework. A philosophy." After: "It is a philosophy: [state it]." The reader needs no runway.

### Rule of three [flattens]
Triads of adjectives, nouns, phrases, examples, or bullets, deployed to look comprehensive. Keep the one or two items that matter, or restructure so no list exists. Applies to lists and headings too.
> Before: "The event features keynote sessions, panel discussions, and networking opportunities. Attendees can expect innovation, inspiration, and industry insights."
> After: "The event includes talks and panels, with time for informal networking between sessions."

### Mirror symmetry [flattens]
Consecutive sentences or paragraphs with identical shape. Give the second thought a different shape, length, or angle.
> Before: "Engineers want clarity. Managers want context."
> After: "Engineers want clarity. For managers it's more about context, the things around the decision you can't see from outside."

### Rhetorical question plus instant answer [performs]
Fold into a statement. A genuine question left open for the reader is fine; the self-answered setup is the tell.
> Before: "What does this mean in practice? It means teams need autonomy." After: "Teams need autonomy."
> Before: "The result? Devastating." After: "The rollout failed in all three regions."

### Colon and dash reveals [performs]
Noun phrase, colon, dramatic payoff. Rewrite as a plain sentence; colons are for lists, labels, and quotes.
> Before: "The detail that makes it work: a separate agent grades it."
> After: "A separate agent does the grading, which is what makes it work."

### Staccato drama [performs] [flattens]
Stacked short fragments manufacturing intensity. One short sentence lands a point; a run of them is engineered.
> Before: "Then AlphaEvolve arrived. It had no preference for symmetry. No aesthetic prior. No nostalgia for human taste. The old rules were gone."
> After: "AlphaEvolve changed the search because it did not favor symmetry or human-looking designs, which made some of the older assumptions less useful."

### Manufactured punchlines and fake-profound kickers [performs]
Every paragraph landing a quotable, or a closing aphorism ("X is the Y of Z", "the currency of", "the architecture of", "X is not a tool but a mirror", "X becomes a trap"). Delete the kicker; do not rewrite it into a better one. End on the clearest concrete sentence already present, or a plain takeaway or next action.
> Before: "Symmetry is the language of trust." After: "Symmetric layouts feel more predictable to users."

### Superficial participle analysis [performs] [evades]
A trailing "-ing" clause faking depth: highlighting..., underscoring..., emphasizing..., reflecting..., symbolizing..., showcasing..., ensuring..., demonstrating..., contributing to..., fostering..., encompassing..., enhancing... Fix: replace with the concrete consequence, or cut. If a named source made the analysis, cite what they actually said; unattributed, it is your opinion wearing a source's coat.
> Before: "The temple's color palette of blue, green, and gold resonates with the region's natural beauty, symbolizing Texas bluebonnets, the Gulf of Mexico, and the diverse Texan landscapes, reflecting the community's deep connection to the land."
> After: "The temple is painted blue, green, and gold, colors meant to evoke Texas bluebonnets and the Gulf of Mexico."
> Before: "The launch adds file search, highlighting the team's commitment to better workflows."
> After: "The launch adds file search, so users can find old drafts without leaving the editor."

### False ranges [performs]
"from X to Y" where no real scale connects the endpoints.
> Before: "Our journey has taken us from the singularity of the Big Bang to the grand cosmic web, from the birth and death of stars to the enigmatic dance of dark matter."
> After: "The book covers the Big Bang, star formation, and current theories about dark matter."

### Copula avoidance and fake-strong verbs [performs]
serves as, stands as, functions as, acts as, operates as, represents, marks, refers to (in definitions), boasts, features, offers, maintains (marketing sense), ventured into X as, began his career as (when it just means was). Fix: is, are, was, has. Simple copulas are one of the strongest human signals; restore them freely.
> Before: "Gallery 825 serves as LAAA's exhibition space for contemporary art. The gallery features four separate spaces and boasts over 3,000 square feet."
> After: "Gallery 825 is LAAA's exhibition space for contemporary art. It has four rooms totaling 3,000 square feet."
> Before (definition lead): "Catchment area refers to the geographic area from which a health facility draws its patients."
> After: "A catchment area is the area from which a hospital or clinic draws its patients."

### False agency [evades]
Inanimate things doing human verbs: the complaint becomes a fix, the decision emerges, the culture shifts, the conversation moves toward, the data tells us, the market rewards, a bet lives or dies. Fix: name the human; if none fits, use "you" or "we".
> Before: "Within a week, the complaint becomes a fix." After: "The team fixed it that week."

### Passive voice and subjectless fragments [evades]
Fix: actor first, active verb. Keep a passive only when the actor is genuinely unknown or irrelevant and the register expects it (a methods section, for instance).
> Before: "No configuration file needed. The results are preserved automatically."
> After: "You don't need a configuration file. The system preserves results automatically."
> Before: "The decision was reached after review." After: "The board decided after reviewing the audit."

### Weasel attribution and overgeneralization [evades]
Experts argue, Industry reports suggest, Observers have cited, Some critics say, widely regarded as, studies show, several publications (when one is cited), one review inflated into "reviewers", "such as" implying a longer list than exists. Fix: name the source and what it actually said, or cut the claim. If the user has no source, ask; never invent one. An unsupported claim gets cut, never decorated.
> Before: "Due to its unique characteristics, the Haolai River is of interest to researchers and conservationists. Experts believe it plays a crucial role in the regional ecosystem."
> After: "Researchers and conservationists study the Haolai River for its unusual characteristics."

### Hedging stacks [evades]
One hedge maximum per claim; better, commit. Distinguish from a writer's single genuine hedge, which stays.
> Before: "It could potentially possibly be argued that the policy might have some effect on outcomes."
> After: "The policy may affect outcomes."

### Lazy extremes [evades]
always, never, everyone, nobody, every single, used as false authority for a vague claim. Replace with the specific case.
> Before: "Everyone struggles with migrations."
> After: "We've all copy-pasted from a migration guide, missed an edge case, and spent two hours debugging why tests fail."
Note the flip side: a definitive claim a human can defend ("was the first", "is the only") is a human signal; keep it when true.

### Interrogative-opener crutch [flattens]
Sentences repeatedly opening with What/Why/How, paragraphs opening with "So", openers like "Look,". Occasional use is human; the repeated crutch is the tell.
> Before: "What makes this hard is the coupling." After: "The coupling makes this hard."

### The "Despite challenges" formula [performs]
"Despite its X, [subject] faces several challenges... Despite these challenges, [subject] continues to thrive as..."
> Before: "Despite its industrial prosperity, Korattur faces challenges typical of urban areas, including traffic congestion and water scarcity. Despite these challenges, Korattur continues to thrive as an integral part of Chennai's growth."
> After: "Korattur has recurring traffic congestion and water shortages."

### Hyphenated-pair uniformity [flattens]
AI hyphenates compounds everywhere, including after the noun. Keep attributive hyphens, drop predicate ones.
> Before: "The team is cross-functional and the report is high-quality."
> After: "The team is cross functional and the report is high quality." (Attributive stays: "a high-quality report".)

## 4. Paragraphs and composition

### A bow on every paragraph [flattens]
Every paragraph resolving into a tidy takeaway. Let at least a third of paragraphs just stop; the reader can sit with an idea.

### Fractal summaries [flattens]
Announce, say, recap: the same content three times at shrinking scale. Say it once.
> Before: "In this section we'll look at caching. [three paragraphs] In short, caching matters at every layer."
> After: [the three paragraphs].

### Summary conclusions [performs]
"In conclusion", "In summary", "Overall", "Ultimately", or a final paragraph restating the piece. The reader was just there. End on the last concrete point, takeaway, or action.
> Before: "In summary, the educational trajectory for nurse scientists involves a progression from a master's degree to a PhD, followed by postdoctoral training. This structured pathway ensures that nurse scientists acquire the necessary knowledge and skills."
> After: (cut; the pathway was just described).

### Generic upbeat send-offs [performs]
> Before: "The future looks bright for the company. Exciting times lie ahead as they continue their journey toward excellence."
> After: (cut the paragraph; end on the last concrete fact, or sourced real plans).

### Template scaffolding [flattens]
Rigid Introduction / Background / Challenges and Future Outlook / Conclusion outlines; a heading followed by a one-line warm-up restating the heading; a thematic break before every heading; a header over every two sentences. Fix: headings as specific statements ("Docs go stale", never "Background"), warm-up lines cut, structure following content.
> Before: "## Performance // Speed matters. // When users hit a slow page, they leave."
> After: "## Performance // When users hit a slow page, they leave."

### Synonym cycling [flattens]
Repetition-penalty artifact. If the clear word is right, repeat it.
> Before: "The protagonist faces many challenges. The main character must overcome obstacles. The central figure eventually triumphs. The hero returns home."
> After: "The protagonist faces many challenges but eventually triumphs and returns home."

### Transition clustering [flattens]
Furthermore, Moreover, Additionally, Consequently, Subsequently, Notably piled up, especially sentence-initial. One "however" is normal; the pile is the tell. Keep at most two formal transitions in the whole piece; often the connection can be implicit, or "And"/"But"/"So" carries it.

### Dilution [flattens]
One point restated across sections, one metaphor beaten to death, stacked historical analogies for borrowed authority ("Apple didn't build Uber. Facebook didn't build Spotify."). One point per section; one use per metaphor; one analogy or none.

### Diff-anchored writing [evades]
Docs or comments narrating the change instead of the thing. Changelogs and migration guides are exempt.
> Before: "This function was added to replace the previous approach of iterating through all items, which caused O(n²) performance."
> After: "This function uses a hash map for O(1) lookups, avoiding the O(n²) cost of naive iteration."

## 5. Formatting

### Inline-header bullets [flattens]
Lists shaped "- **Term:** description", every item bolded, often in threes. Convert to prose, or to plain bullets when a list genuinely helps.
> Before: "- **User Experience:** The user experience has been significantly improved. - **Performance:** Performance has been enhanced through optimized algorithms. - **Security:** Security has been strengthened with end-to-end encryption."
> After: "The update improves the interface, speeds up load times through optimized algorithms, and adds end-to-end encryption."

### Abstract bullets [evades]
List items that are bare category nouns ("Enhanced productivity", "Improved efficiency", "Better collaboration") carry no information. Make each item a concrete claim or cut the list.
> Before: "Benefits: 1. Enhanced productivity 2. Improved efficiency 3. Streamlined workflows"
> After: "This saves time two ways: edge cases ship inside the package instead of living in docs nobody reads, and the tests are generated rather than written."

### Mechanical boldface [performs]
Bold sprinkled mid-sentence for emphasis, or every instance of a chosen term bolded. Cut nearly all bold; content carries emphasis.

### Emoji as formatting [performs]
Emoji decorating headings or bullets. None, unless the user asks.

### Title Case Headings [flattens]
> Before: "## Strategic Negotiations And Global Partnerships" After: "## Strategic negotiations and global partnerships"

### Em and en dashes [flattens]
The most notorious style tell, especially spaced ( — ) in punched-up sales rhythm, and double hyphens ( -- ) used the same way. Short copy: zero. Long form: at most two, and only where a dash clearly beats the alternatives. Replace each, in order of preference: period, comma, colon, parentheses, restructure. Exception: a user's writing sample that uses them; match the sample's rate, since matching the author beats scrubbing the tell.
> Before: "The new policy — announced without warning — affects thousands of workers."
> After: "The new policy, announced without warning, affects thousands of workers."

### Structure over-formatting [flattens]
Bullets where two sentences of prose read better; tiny two-row tables that should be a sentence; unicode arrows and decorative symbols; markdown syntax leaking into contexts that do not render it (asterisks and ### in a plain-text field, hallucinated wiki templates); skipped heading levels. Format follows content; convert to the context's native format.

### Quote-mark inconsistency [flattens]
Curly and straight quotes or apostrophes mixed in one text. Normalize to the context's convention. Curly quotes alone prove nothing (Word, macOS, and most CMSes auto-curl); the inconsistency is the signal, and only alongside other tells.

## 6. Evidence and citations

### Vague declaratives [evades]
"The implications are significant." "The reasons are structural." "The stakes are high." "This is genuinely hard." A sentence announcing importance without naming the thing. Replace with the specific implication, reason, or difficulty, or cut.
> Before: "The tool significantly improves engineering productivity."
> After: "The tool cut review time from 30 minutes to 8."

### Regression to the mean [evades]
The mechanism behind everything here: specifics smoothed into categories. Protect every existing specific absolutely; restore specifics only from the source or the user (Law 1). The anywhere-test: if a sentence could appear unchanged in any piece on any topic, it says nothing; replace it with one only this piece could contain.
> Before: "The initiative faced financial challenges." After: "We lost $40k on that campaign."

### Notability piling [performs]
Parades of outlets, "independent coverage", "trade publications", "profiled in multiple high-quality outlets", "maintains an active social media presence". Keep the one or two citations that carry real context (who said what, where); cut the parade. Do not invent the context to make the trimmed version sound better.
> Before: "Her views have been cited in The New York Times, BBC, Financial Times, and The Hindu. She maintains an active social media presence with over 500,000 followers."
> After: "Her views have been cited in The New York Times and the BBC."

### Citation integrity [evades]
Flag, fix, or remove: references that cannot be verified, invalid DOIs and ISBNs, dead links, book citations with no page numbers, placeholder access dates (2025-XX-XX), tracking artifacts (utm_source=chatgpt.com), and leaked model markup (oaicite, turn0search, contentReference, [cite: 1], grok_card, [span_1]). Never fabricate a replacement reference.

## 7. Tone

### Sycophancy and over-enthusiasm [performs]
"We're thrilled to announce", "Great question!", "You're absolutely right!", exclamation clusters. Lead with information.
> Before: "We're excited to announce a revolutionary new approach to migrations!"
> After: "We built X because Y kept breaking."

### Press-release voice [performs]
> Before: "CEO Allan Kilavuka emphasized the airline's commitment to sustainability, customer focus, and Africa's prosperity through responsible corporate practices."
> After: "The airline says the projects will cut fuel use; Kilavuka has tied them to its 2030 emissions target." (Only if the source supports it; otherwise cut.)

### Both-sides mush [evades]
Perfectly balanced perspectives with no position anywhere. Take the stance the content supports; where genuine disagreement exists, name who disagrees and why.

### Patronizing pedagogy [performs]
"Think of it as...", "Let's break this down", "imagine you're a...", deployed for an audience that needs none of it. Teach only when the reader is actually a beginner.

### Invented concept labels [performs]
A fresh coinage presented as established: "the supervision paradox", "the visibility trap". Describe the phenomenon plainly, or introduce the coinage honestly, once.

### False vulnerability [performs]
Unearned intimacy: "I promise", "trust me", "we've all been there", "They exist, I promise", drama verbs like "creeps in". Cut.

## 8. Detection guidance

For the detect job, and for restraint while editing.

**Clusters convict; singles do not.** One em dash means nothing. Em dashes plus rule-of-three plus "vibrant tapestry" plus a Conclusion section is a confession. A pronounced style shift partway through a text, or text that stops mid-sentence (a mostly historical artifact of token limits), count toward a cluster. Report the cluster, not a verdict.

**Never flag on these alone.** Perfect grammar (professionals exist). Mixed casual and formal registers (common in technical people, young writers, neurodivergent writers, multi-author pages). Bland prose without specific tells (dry writing is legal). Fancy or academic vocabulary in general (AI overuses specific words, and their synonyms are innocent). One transition word. Curly quotes alone. Em dashes alone (journalists love them). One short emphatic sentence. Unsourced claims (most of the web is unsourced). Salutations and sign-offs. Clean complex formatting.

**Preserve these human signals; over-editing them destroys the piece.** Simple is/has phrasing. Plain words where a stiffer synonym exists (used, wrote, died, moved, tried). Definitive claims the writer can defend ("was the first"). Genuine hedges and intensifiers (very, perhaps, tends to). Specific, unusual, hard-to-fabricate detail ("the lawyer who used to work upstairs from my dentist"). Mixed feelings left unresolved. Era-bound slang and in-jokes. Self-interruptions, parentheticals, asides. Uneven rhythm. Wordy human constructions (as a result of, the fact that) in casual registers. The writer's ability to explain their own choices is the strongest human signal of all: when in doubt, ask before accusing.
